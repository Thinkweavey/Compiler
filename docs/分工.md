## 成员A（词法分析）Issue 清单

### Issue A1：实现词法主流程
- **负责文件**
  - `src/main/java/edu/groupname/compiler/lexer/LexerImpl.java`
  - `src/main/java/edu/groupname/compiler/lexer/Lexer.java`
- **接口**
  - `LexicalAnalyzerResult analyze(String sourceCode)`
- **任务**
  - 扫描源码并生成 token 序列，保证 token 顺序与源码一致。
  - 输出必须满足 A->B 契约：`LexicalAnalyzerResult.tokens()` 末尾仅追加 **1 个** `EOF`。
  - `Token.lexeme()` 保留源字面值，供 B 侧按 `lexeme` 做终结符映射（关键字/运算符/分隔符）。
- **DoD**
  - 输入 `{ int a; a = 1; }` 至少输出 `{/int/id/;/id/=/num/;/}` 对应 token + EOF。
  - 多次调用 `analyze` 不会重复叠加 EOF，始终只有 1 个结束符。
  - `errors` 为空时 `hasErrors()==false`。

### Issue A2：补齐 token 分类与关键字识别
- **负责文件**
  - `src/main/java/edu/groupname/compiler/common/token/TokenType.java`
  - `src/main/java/edu/groupname/compiler/common/token/KeywordTable.java`
  - `src/main/java/edu/groupname/compiler/common/token/Token.java`
- **接口/点位**
  - `KeywordTable.isKeyword(String lexeme)`
- **任务**
  - 区分关键字、标识符、整数/实数、布尔字面量、运算符、分隔符。
  - 严格遵循 A->B 映射约定：
    - `IDENTIFIER -> id`
    - `INTEGER_LITERAL -> num`
    - `REAL_LITERAL -> real`
    - `BOOLEAN_LITERAL(true/false) -> true/false`
    - `KEYWORD/OPERATOR/DELIMITER -> 直接使用 lexeme`
    - `EOF -> 解析器侧 $`（由 B 内部映射）
  - 保证 `bool` 识别为 `KEYWORD`，`true/false` 识别为 `BOOLEAN_LITERAL`。
- **DoD**
  - `if/else/while/do/break/int/float/bool/true/false` 识别正确。
  - 运算符至少覆盖：`+ - * / = == != < <= > >= && || !`。
  - 分隔符至少覆盖：`{ } ( ) [ ] ;`。
  - `program` 是否保留字与 B 同学文法约定一致。

### Issue A3：注释、空白、位置信息与词法报错
- **负责文件**
  - `src/main/java/edu/groupname/compiler/common/position/SourcePosition.java`
  - `src/main/java/edu/groupname/compiler/common/error/LexicalError.java`
  - `src/main/java/edu/groupname/compiler/lexer/LexicalAnalyzerResult.java`
- **任务**
  - 跳过空格、`\n`、`\t`、注释（单行/块注释按小组约定）。
  - 维护行列号，非法字符进入 `LexicalError`。
  - 注释与空白不得产出 token（严格符合接口文档第 3.3 节）。
  - 非法词素/字符错误信息包含可定位的 `line/column`，未知位置使用 `SourcePosition.UNKNOWN`。
- **DoD**
  - 错误信息含行列号。
  - 至少 5 组词法测试：正常、含注释、含非法字符、布尔/关键字混合、实数与整数混合。
  - 词法报错后依然返回可安全读取的 `LexicalAnalyzerResult`（供 D 在 `CompilerPipeline` 执行短路）。

    - 关键字与布尔字面量分流（`bool` vs `true/false`）
