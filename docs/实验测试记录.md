## 实验一：测试组 8 — 词法错误（非法字符）

**输入**（`samples/sample8_lexical_error.src`）：

```text
{ int @a; }
```

**说明**：`@` 不是文法合法字符，词法分析进入 ERROR 状态，记录错误后继续扫描。

(}, 分隔符)
--- 符号表（按出现顺序） ---
