# 词法输出样例清单（A -> B 联调）

以下样例基于 `Token.lexeme()` 序列展示，末尾统一追加 `<EOF>`，供语法分析阶段映射到 `$`。

## 样例1：基础声明与赋值

- 输入：`{ int a; a = 1; }`
- 输出：`{ / int / a / ; / a / = / 1 / ; / } / <EOF>`

## 样例2：布尔类型与字面量分流

- 输入：`bool flag; flag = true; flag = false;`
- 输出：`bool / flag / ; / flag / = / true / ; / flag / = / false / ; / <EOF>`
- 说明：`bool` 是 `KEYWORD`，`true/false` 是 `BOOLEAN_LITERAL`。

## 样例3：运算符与分隔符

- 输入：`if (a >= 10 && b != 0) { a = a + 1; }`
- 输出：`if / ( / a / >= / 10 / && / b / != / 0 / ) / { / a / = / a / + / 1 / ; / } / <EOF>`

## 样例4：注释与空白跳过

- 输入：
  ```text
  // one line
  { /* block */ int a; }
  ```
- 输出：`{ / int / a / ; / } / <EOF>`
- 说明：空白与注释不产出 token。

## 样例5：整数与实数

- 输入：`a = 1; b = 2.5;`
- 输出：`a / = / 1 / ; / b / = / 2.5 / ; / <EOF>`
- 说明：`1` 为 `INTEGER_LITERAL`，`2.5` 为 `REAL_LITERAL`。
